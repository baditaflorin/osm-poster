OSM Poster — Quick Reference

Browser-only. Your design lives in the URL hash and localStorage.

Keyboard Shortcuts

RRandomize palette EExport with current settings FToggle fullscreen preview MToggle center marker [Previous style preset ]Next style preset ?This dialog ⌘ ZUndo ⌘ ⇧ ZRedo EscClose dialog / clear search

Sharing

Every change updates the URL hash. Copy the URL to share your exact design — palette, layers, view, caption, marker, even the random seed.

GPX Routes

Upload a .gpx file (run, ride, hike) and the route renders in the accent color. Files never leave your browser.

Reproducible Random

Type a seed before pressing Randomize for a deterministic palette. The seed is shared via URL so anyone opening the link sees the same colors.

Pinhead Icons

Curated picks from Pinhead — 1,000+ public-domain map icons. Click to use as your marker.
